Calculadora de Média Semestral
Programa em Python que calcula a média semestral de um aluno com validação de notas.
Funcionalidades

Lê duas notas com validação (aceita apenas valores entre 0 e 10)
Calcula a média semestral
Exibe a situação do aluno: Aprovado, Recuperação ou Reprovado

Regras
MédiaSituação>= 7.0Aprovado>= 5.0Recuperação< 5.0Reprovado
Como executar
bashpython calculadora.py
Exemplo de uso
========================================
   CALCULADORA DE MEDIA SEMESTRAL
========================================
Nome do aluno: Marcos
Digite a nota 1: -3
Nota invalida! Digite um valor entre 0 e 10.
Digite a nota 1: 8.5
Digite a nota 2: 7.0

========================================
  Aluno: Marcos
  Nota 1: 8.5
  Nota 2: 7.0
  Media:  7.75
  Situacao: APROVADO
========================================
